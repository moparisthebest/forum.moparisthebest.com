Then like The Un-Named said, type in your password, and after that it should process that command.

And also Bang Train was correct as well. Every time you restart your computer, you should see the Grub Menu which is a menu displaying the hard drives that are available to you. If you want to change around the Grub Menu on bootup, like get rid of some entries, add entries or whatever.

Here is the command you use to edit the file that shows you the HDs on boot up in Grub Menu.
The first command will back up your menu.lst file and the next will let you edit that file. Have fun and be careful!

sudo cp /boot/grub/menu.lst /boot/grub/menu.lst_backup
gksudo gedit /boot/grub/menu.lst
