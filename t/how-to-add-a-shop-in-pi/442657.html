Configure the shop items in data/shops.cfg

The in ActionHandler under the void [tt]firstClickNpc(int)[/tt] add [tt]c.getShops().openShop(#);[/tt]

The # being the number you added inside the shop file before the name. I.E: [tt]shop = #[/tt]
