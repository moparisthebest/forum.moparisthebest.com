Here is a list of common jaghash values, if the value you are looking for is not listed you can use the applet on the bottom to convert a string to jaghash value.

http://silabsoft.org/?list=jaghash

Credits: Bentski first released a application that contained this information refactored.

the applet source:

public class JagHashCreator extends Applet implements ActionListener {
    private TextField input;
    private TextField output;
    private Label label1;
    private Label label2;
    private Label label3;
    private Label label4;
    private Button b1;

    public void init() {
        this.setLayout(null);
        label4 = new Label("String to Jagex Hash Converter");
        label4.setBackground(Color.black);
        label4.setForeground(Color.blue);
          label4.setBounds(65,0,200,13);
        this.add(label4);
        label1 = new Label("Enter Text: ");
        label1.setBackground(Color.black);
        label1.setForeground(Color.blue);
        label1.setBounds(20,20,75,13);
        this.add(label1);
        input = new TextField(20);
        input.setBounds(100,16,100,20);
        this.add(input);
        label2 = new Label("Jagex Hash:");
        label2.setBounds(20,45,75,13);
        this.add(label2);
        label2.setBackground(Color.black);
        label2.setForeground(Color.blue);
        output = new TextField(20);
        output.setBounds(100,41,100,20);
        this.add(output);
        b1 = new Button("Convert");
        b1.setBounds(215,16,50,20);
        this.add(b1);
        label3 = new Label("Made by Silabsoft");
        label3.setBounds(100,70,150,13);
        label3.setBackground(Color.black);
        label3.setForeground(Color.blue);

        b1.addActionListener(this);
        add(label3);
        this.setBackground(Color.black);
    }

    public void actionPerformed(ActionEvent ae) {
          output.setText(""+stringToJagexHash(input.getText()));
    }

    public int stringToJagexHash(String s) {
        int i = 0;
        s = s.toUpperCase();
        for (int j = 0; j < s.length(); j++)
            i = (i * 61 + s.charAt(j)) - 32;
        return i;
    }
}

What does the client do with this hash? parse the jagfolders for the specific data then returns the decompressed data.
