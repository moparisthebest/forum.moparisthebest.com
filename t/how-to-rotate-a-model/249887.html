cloudscape July 31, 2016, 3:54am #3

For objects to change to way they are facing you use this part of the code:

makeGlobalObject(XXXX, YYYY, ID, Face, Type);

The face is the direction that the object faces, the values for this are, 0,-1,-2,-3
I will let you work out which is which

Eg code.

makeGlobalObject(XXXX, YYYY, 1000, -3, 10);

The last number always has to be 10.

Hope this helped
