public static int rewardObject() {
    return rewardObject[(int) (Math.random() * rewardObject.length)];
}

This is a method, this method cannot go inside another method.

public static int rewardObject[] = {ItemID, ItemID};

This is a variable, this variable cannot go inside of a method. (It can if you take away the public static though)

case #:
    addItem(rewardObject(), 1);
    break;

This is a case, this has to go inside of a switch statement

switch (objectId) {

This is the switch statement that the case must go in.
