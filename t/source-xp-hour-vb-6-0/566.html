Private Sub cp_Click()
Dim ipt As String
Dim ips As String
ipt = InputBox("Creatures Hp?")
ips = InputBox("Ur kills per minute?")
label1 = ipt * ips * 4 * 60
MsgBox "U get " &  label1.caption & " Xp/Hour"
End Sub

Add an command button called Cp and a label called label1. Then you're finished. It will ask the player what is the creatures hp and the kills per minute. Finito, you just made an calculating for getting the xp in 1 hour :eek:. Got any questions ask it.

BTW: Label1 has to be Label1.Caption or it won't display anything.

If you'd wan't to add it to a module, to calculate you could change it like this:

Function XP_Hour(CreatureHP As Long, KillAM As Long) As Long
    XP_Hour = CreatureHP * KillAM * 4 * 60
End Function

That would be easier to use in programs.

In this case:

Private Sub cp_Click()
Dim ipt As String, ips As String
    ipt = InputBox("Creatures Hp?")
    ips = InputBox("Ur kills per minute?")
    Label1.Caption = Str(XP_Hour(Val(ipt), Val(ips))
    MsgBox "U get " &  Label1.Caption & " Xp/Hour"
End Sub
