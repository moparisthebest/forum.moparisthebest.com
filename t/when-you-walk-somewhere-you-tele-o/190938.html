There are two simple ways.

One, you can find the method that involves walking, and after the character has stopped moving, check its coordinates with an if statement. Something like

if(absX == 1000 && absY == 2000)
{
      teleportToX = 1;
      teleportToY = 2;
}

I forgot how to teleport because I haven‚Äôt opened one of those up for a long time. So look into that.

The second way is the constantly check each players location every half second. This will have the same result as the first option, but will create much more lag than needed. All you have to do is stick that if statement above into boolean process().

I reccomend the first option.
