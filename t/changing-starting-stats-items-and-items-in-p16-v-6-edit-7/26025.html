Difficulty: 2

Assumed Knowledge: know how to change things to what you like or copy and paste

Classes Modified: Player.java

step 1 go into player.java and search for this

		for (int i=0; i<playerLevel.length; i++) { //Setting Levels[/code]

you will see something like this

[code]			if (i == 3) {
				playerLevel[i] = 10;
				playerXP[i] = 1100;
			} else {
				playerLevel[i] = 1;
				playerXP[i] = 0;
			}
		}[/code]

[b]step 2[/b]

now change the 10 to whatever lvl you want to start at then the 1100 to the exp at that lvl. this is for hp but to add more you can change it so it looks like this

[code]			if (i == 3 || i == 2) {
				playerLevel[i] = 10;
				playerXP[i] = 1100;
			} else {
				playerLevel[i] = 1;
				playerXP[i] = 0;
			}
		}[/code]

now all i added in there was this [code] || i == 2[/code] the 2 stands for the skill number so that is str 3 is hp i think. to add more just do the same thing do [code] || i == #[/code] then change the # to the skill number you want to be higher when you start.
