he already knows about ImageIO, he wants to know about editing the image he has

the read() method in ImageIO gives you a BufferedImage instance, then you should call getGraphics() which gives you a Graphics instance, and you can use that to draw on it, change the colours of pixels, when you‚Äôve finished editing the image, you can save it again with ImageIO.write()

this code draws a red line on the image and saves it again

import java.awt.image.BufferedImage;
import java.awt.Graphics;
import java.awt.Color;
import javax.imageio.ImageIO;
import java.io.File;

//method decleration
BufferedImage image = ImageIO.read(new File("image.png"));
Graphics g = image.getGraphics();
g.setColor(Color.RED);
g.drawLine(10, 10, 200, 200);
g.dispose();
ImageIO.write(image, "png", new File("edited-image.png"));
