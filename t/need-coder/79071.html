Starter Kit:
Put this with your other item-spawn codes:

else if (command.equalsIgnoreCase("starter")) if (hasegg == 0) { hasegg = 1; addItem(ITEMID, AMOUNT); }

Just replace ITEMID with the ID of the item you want spawned, and AMOUNT with the amount of items you want spawned. If you want more than one item spawned that is not stackable, then use the note form of the item, for example, the code for Blue Phat is 2242, but the note for is 2243. The note form of an item is always 1 number after the regular form of the item.
