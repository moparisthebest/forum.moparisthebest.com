  1. Check the modeling section of the forums, or use the search bar.

  2. If you are using a 317 server, Go into your client.java file and hit ctrl + f.

Then type in

public boolean nonWild

Then when you see

|| (absX >= #### && absX <= #### && absY >= #### && absY<= ####)

You are in the right section. You then have to add in your own coordinates on what you want to be a safe zone.

Pretty much what

 absX >= #### && absX <= ####

is saying is that any X value between the two numbers is safe.

Ex:

 absX >= 3493 && absX <= 3500

^ That means that any X-value between the value of 3493 and 3500 will be safe. Then you need to put in your Y-values so that you don’t just have a strip going across your map that is safe. So in fact, you are basically making a box for your safe area.

                       [v absY <= ####]
                       --------X---------
                       |                       |
                       |                       |

[^ absY >= ####]
