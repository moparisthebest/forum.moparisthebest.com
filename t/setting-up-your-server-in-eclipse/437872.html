I want people to move on from using Notepad++ and plain old notepad and get the hang of using an IDE. I couldn‚Äôt find any detailed tutorial(s) on using eclipse/netbeans/etc so I decided to make one

Note: I am using Ubuntu, but apart from getting the installer everything else will be the same as windows

[size=12pt]1. Downloading Eclipse[/size]

First you will need to click here and you will see this page.

Download ‚ÄòEclipse IDE for Java Developers‚Äô. When its done, start the installer (windows).
If you are running linux, download the tarball and use this if you dont know how to install it.
sudo apt-get install eclipse is as easy

Okay so the installer is pretty straight forward, just install it

[size=12pt]2. Moving your source to workspace folder.[/size]

Pretty straight forward. Cut or copy your source and navigate to your workspace. My workspace folder is in home/pc/workspace. Windows would probably be C:/Users//workspace or C:/Documents and Settings/_/workspace.

[size=12pt]3. Creating a project[/size]

Launch the program. The splash screen should show up looking like this:

When you first start up, it will ask for your workspace folder. Dont modify it just click okay.

When its started, you will see this:

Hover over the orange project explorer area (top left corner) and right click>New>Project.

Choose java project and click next.

Now make the project name in the top box your source folder name. It should automatically pick up most of the other settings and then click Finish.

[size=12pt]4. Editing your server[/size]

Its easy to edit your server. Just use the project explorer to navigate to the file you want to modify and double click it.
Errors will show up in the bottom box under the ‚ÄòProblems‚Äô tab. Warnings dont affect the server updating when you save, dont worry

Note: If the problems box doesnt show up, go to Window>Show view>Problems

[size=12pt]5. Running your server[/size]

Right click your original project folder, go to Run as and then click 2. Java application. It will ask you to choose the main class (usually server). Click server and then Ok.

You should now get something like this in your Console box


(To stop the server when you want to restart, push the red square to stop and then go Run>Run last launched)

Congratulations! You can now use eclipse (Basically)
