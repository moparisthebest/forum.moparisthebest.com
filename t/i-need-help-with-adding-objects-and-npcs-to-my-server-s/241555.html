Below the last one, add this‚Ä¶

makeGlobalObject(xxxx, yyyy, -x, 10); // item name

xxxx = X coordinates

yyyy = Y coordinates

-x = Position the item is facing. 0 is north, -1 is east, -2 is south, -3 is west.

10 = Height level i think, just keep it at 10

item name = What ever the item name is your adding.
