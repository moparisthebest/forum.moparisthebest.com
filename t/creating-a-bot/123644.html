There are a few different kind of bots, depending on which one you want to make, you will need to focus on learning certain things or others.

If you want to make a color clicker, such as Scar, you can use basically any language, but you will probably need to learn something about image processing (to find deformed bitmaps, etc.).
If what you want is a bot aryan style, which makes use of deobs, you will need to learn java and java debugging.
If what you want is a BCEL bot or similar, which ‚Äúhooks‚Äù methods, or autoupdates, and all that, you will need to learn java and java debugging too, and java bytecode, BCEL, etc.
Finally, if what you want is a packet or proxy bot (my favourites) you will need to know a programming language (which can be java), java, java debugging, and you could probably use some knowledge of BCEL / Javaagents / Regex or Parsers for auto-updating.

For people to be able to create scripts, you will need to use a scripting engine. I don‚Äôt know which engines are available for Java, but in C++ you could use, Python, Lua, or many others.

Note: I don‚Äôt know BCEL or java bytecode myself so the information about hooking bots is probably kind of general. In any case if you have any question, feel free to contact me in irc
