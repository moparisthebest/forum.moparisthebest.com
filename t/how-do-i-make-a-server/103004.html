Ok, Get hamachi, www.hamachi.cc
Get a source http://www.moparisthebest.com/smf/index.php/board,49.0.html

Create a hamachi network (do this by opening hamachi and clicking create a network)
Go into your Moparscape files (when you downloaded moparscape where you saved them is where the files are)

delete all files in Hybridscape and replace with the files you got from the download section

click compile.bat, press c

once it is done compiling exit out and click runserver.bat

open moparscape and for the server ip there will be something at the top of hamachi, Thats your IP, Type that in where it says Server IP:

Type in your wanted username/password, Now your all set, Anyone can join…
