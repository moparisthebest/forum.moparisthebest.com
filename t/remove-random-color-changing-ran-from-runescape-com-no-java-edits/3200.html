Its simple, if you have the 1.5.06 of the jre all your runtime classes are in rt.jar, along with the precious math.class with its random() function, so I extracted and decompiled math.class and made the random() method return a constant (0.3), then compiled and with winrar overwrote the old math.class

when running runescape, the flames dont look right, and colors on the main screen along with color and location of the mm symbols never change, even if you close the client and open another one, the road color however still changes randomly, so one must assume it is sent from the server (they arent using java.util.random because I also edited that entire class to return constant values, yet they still change)

here I am attaching my edited math.class and math.java, you simply have to open rt.jar (located in your java/lib folder) with winrar, navigate to java/lang/ and replace the old math.class with this math.class (wouldnt hurt to keep a backup)
