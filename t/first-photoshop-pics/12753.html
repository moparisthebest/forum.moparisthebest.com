Ok, some constructive critisiscm here:

Picture 1 - Good blending in with the picture but you have the colour all wrong. The colour should mostly compliment eachother. (Look at a colour wheel for more information.) Although, they do not always have to compliment eachother. It’s just the way you use them. In that picture, you have used them the wrong way. The pencil marks are WAY too big. Try using a 1-3 pixel brush. The text is good.

Picture 2 - It looks like you have blended C4Ds. This is a good look. Those two colour do suite eachother. Although, you need to focus on blending the duplicated picutres a bit more. I can see them very clearly. Try and make them almost invisible.

Picture 3 - It is ok, but you havent used to overlay correctly, I don’t know what you done (:P) but it doesn’t seem to work out. Again, it seems as though you have used the wrong colours.

Overall - Focus on using the colour a lot more betterly (sp, lol). Also, try looking in to a few blending tutorials. Try www.pslover.com for some good ones.
