Left panel:-

Starting from the top, we have a refresh button. This refreshes the directory listing for the current directory.

Then we have the directory listing itself.

Then we have 5 checkboxes, fill, edges, points, grid, and axis. These enable/disable the displaying of these items withing the 3D window.

The Cull faces checkbox enables/disables backface culling. Disable this when inspecting your model.

Bkg and Edge color boxes - these allow you to change the color of the background and edge color within the 3D window.

Dropdown menu - For Model veriosn 3 only. This menu allows you to select very basic anim data to be included in your model. The only really useful items are head objects(hats), left hand(shields), right hand(standard weapons(swords) and gloves(daggers,axes,gloves).

Use Blank PRI checkbox - For model version 3 only. This allows you to insert blank priority numbers into the dat file.(limited usefulnedd - for advanced modellers).
