xryz indeed you should use the object tag e.g.

<object type="application/x-java-applet">
  <param name="code" value="randomNumber" />
  <param name="archive" value="randomNumber.jar" />
  <param name="java_arguments" value="-Xmx10m" />
</object>

the above is one of listed ways to embed a java applet in an HTML5-compliant way
