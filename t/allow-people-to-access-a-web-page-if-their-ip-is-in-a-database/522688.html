    $con = mysqli_connect('localhost', 'root', 'password', 'database');

If I remember correctly, it would be faster to use: [code=sql]SELECT COUNT(*) FROM ips WHERE allowed = "127.0.0.1"[/code] As a query, then simply use mysqli_result in PHP to check if it's greater than one, rather than using PHP to check the row count. If you're dealing with a query that could potentially return a lot of large results, it's really wasteful for MySQL to send them to PHP just to have PHP count them. Not only that but the SQL engine can make further optimizations and just serve its own indexed size that it computes internally.
