-> Firefox
+ Very easy to customize and style
+ Internet Options very simple to manage
+ A nice Bookmarks toolbar
+ Able to install many many extensions which are on the mozilla.org website and are checked
+ VERY VERY VERY SECURE!
+ Big fixes and a nice forum to help you with any trouble you "think" you face with firefox
+ Uses fewer resources than IE
+ Built in Search Engines for easy searching with Google, Yahoo!, and many more, and you can even add more search engines
+ Superb popup block feature
+ Able to block images from websites so you can get rid of those annoying fastclick.net and many other ads
+ Can open new windows in tabs, so you don't have to have 500 windows open anymore
+ Faster Browsing which is possible because of improvements to the engine that powers Firefox deliver more accurate display of complex Web sites, support for new Web standards, and better overall performance
+ Clear Private Data very quickly by doing Ctrl+Shift+Del
+ Live Bookmarks update themselves automatically with the latest content from the Web
+ Multi Language Support
- Can't do Windows Updates
