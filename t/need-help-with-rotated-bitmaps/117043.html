Nashi July 30, 2016, 11:40am #2

[code]function FindBitmapRotated(bitmap: Integer; var x, y, Angle: Integer; x1, y1, x2, y2: Integer): Boolean;
Works like FindBitmap, but rotates the bitmap after each search by 1 degree and returns the angle
of the found bitmap afterwards.

function RotateBitmap(bitmap: Integer; angle: Extended): Integer;
Rotates bitmap by specified angle in radians (Pi = 180˚);[/code]

Those functions might help?
