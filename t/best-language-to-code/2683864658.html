Nowadays my favourite languages are C++, C#, Python and Lua

(Disclaimer: This is a personal opinion and I do not make any warranty, expressed or implied, for the accuracy, completeness, or usefulness of this information)

  • C++ (Not to confuse with C): Very powerful languge, high execution and memory efficiency, but low compilation speed and steep learning curve.

  • Python: Dynamic scripting / programming language, fast theorethical dev times but very bad IDEs. Not bad for embedding.

  • Lua: Not good as a stand-alone language, but very good for embedding.
