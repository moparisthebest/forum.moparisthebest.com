Javacisnotrecognized July 29, 2016, 11:27pm #5

Yes, its like this:

The server uses the port going out,
The client uses the port going in.

You have another application using the port. Even if you close the application, it can still be running and using the port. Even if you end it from the task manager, the java.net.Server &java.net.Client sockets will still be running. Solution: Dissconnect all internet services you have entirely and KEEP them dissconnected.
