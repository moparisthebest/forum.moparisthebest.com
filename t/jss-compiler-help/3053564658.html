Dynamic languages such as Javascript are kinda hard to compile to native, you might wanna switch language, or use a Javascript subset, or use some kind of bytecode (either an existing VM or your own). Also, you might wanna get some (e)book on compiler building, there are quite a few around.

Jython July 31, 2016, 11:33am #24

A compiler consists of (at the least) a lexer (lexical analyzer) which converts a character sequence into a stream of tokens, a parser which takes a stream of tokens and builds a concrete syntax tree, and an emitter that emits the concrete syntax tree nodes into an output format, which could be native object code, virtual machine bytecode, or even another language (these are usually called something like language converters or something).
