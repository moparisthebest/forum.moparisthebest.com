tobias222 August 4, 2016, 2:22pm #9

Personally, ovh is very cheap, you get what you pay for.

In my experience, running a server is all about the uplink (bandwidth/speed). More than it is about the RAM anyway. Any server source ive hosted or messed about with sits at 300~mb RAM usage on a linux distro. Never goes above 512. You should be safe with 1G memory. As for uplink, 1Gbps is good but i’ve seen other places with more for cheap.

Saying that, if you are really serious about running on a professional gaming server:
http://www.soyoustart.com/en/innovation-zone/game-servers/

Is the best value price i’ve seen.

Check out lowendbox occasionally for some great offers.
