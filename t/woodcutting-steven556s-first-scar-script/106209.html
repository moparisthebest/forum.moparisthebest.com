Well, theres alot wrong.

First of all, these coordinates might not always be the same, so using them is highly discouraged.
second of all, Movemouse and clickmouse itself is an auto-ban, and having no random points to click makes it all the worse.
Lastly, all you did was movemouse, wait, clickmouse, wait, movemouse, etc.
Try making that into a procedure, so that your main loop can look less cluttered.

Sure it may compile, but theres not much too it. It‚Äô a good try for a first script, as everybody starts somewhere. Now that you have the basic setup of scar in your head, try going deeper and using more than two commands. Also, variables and constants (especially) can come in handy when writing out scripts.

Try using SRL includes, such as Mouse or MMouse for anti-ban.
