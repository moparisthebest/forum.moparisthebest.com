Snippet: PHP to load all CSS and JS files from directory into html

General Programming
SilentCJ August 4, 2016, 1:41am #1

This sort of spawned when I was writing up a jquery slideshow to read some images, nothing too special but here:

	$css_array = glob('css/*.css');
	$length = count($css_array);
	for ($i = 0; $i < $length; $i++) {
		echo '<link href="' . $css_array[$i] . '" rel="stylesheet" type="text/css" />' . "\n\t";
	}

Obviously change the directory to your css directory, and rename variables as you see appropriate.

For the scripts, change it to .js suffix, change variable names and use:

echo '<script src="' . $js_array[$i] . '" type="text/javascript"></script>' . "\n\t";

Instead of the echo statement.
