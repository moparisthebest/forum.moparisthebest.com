Lol. It’s easy.

This is the beginning of any command.

if (command.equalsIgnoreCase("Your command name here"))

It can be in different ways to suit the rights of different players. So in the middle of the last two parentheses you can add this:

playerRights >= 1

This means that mods or higher can only use this command. Remember 1 is for Mods 2 for Admins and 3 for Owner. Just leave as )) if you want anyone to use it.

This is the bulk of the code. It will show what your command will do.

{ additem("Item ID, Amount of that item"); teleTo("X coordinate, Y coordinate") sendMessage("Message to send to the person when they use the command") }

addItem("");

This piece information will allow the command to add an item to the inventory of the command user.

teleTo("");

This piece information will allow the command to teleport a player to the selected coordinates of the command.
It uses X and Y coordinates because RuneScape is set up as a grid.

sendMessage("");

This piece information will allow the command to send a message to the command user.

So the entire command should look like this when you’re done.

if (command.equalsIgnoreCase("Your command name here")) { additem("Item ID, Amount of that item"); teleTo("X coordinate, Y coordinate") sendMessage("Message to send to the person when they use the command") }

It is your choice what you want your command to do. If you want it to do all three things at once then put them all in the command. If you want it to do just one then leave it with one thing.
