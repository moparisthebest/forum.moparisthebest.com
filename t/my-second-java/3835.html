recursion is simply calling a method over and over again through the use of Stacks.

example:

public class recursion {

	/**
	 * @param args
	 */
	public static void main(String[] args) {
		backwards(251);
	}
	public static void backwards(int num){
	      if(num < 10){
	          System.out.print(num);
	      }
	      else{
	    	  System.out.print(num%10);
	    	  backwards(num/10);
	      }
	}

}
