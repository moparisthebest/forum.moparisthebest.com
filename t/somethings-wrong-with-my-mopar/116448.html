Set your variables. Go into:
Start>>My Computer>>View system information>>Advanced>>Environment Variables
Make a new variable and name it PATH. On the second line put: C:\Program Files\Java\jdk1.5.0_09\bin
Make a new variable and name it CLASSPATH. On the second line put: CLASSPATH=C:\Program Files\Java\jdk1.5.0_09\bin;%CLASSPATH%;

Save it and hit ok.
