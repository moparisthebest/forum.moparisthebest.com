PvP icon isn't centered with 108 - 126

Games
Runescape
Economical_Disaster August 4, 2016, 1:20pm #1

How would I go about centering the PVP and safe icon that are both right above the 108 - 126?

EDIT:

Fixed

RSInterface -> PVPInterface, PVPInterface2, PVPInterface3 -> the first setbounds() -> setBounds(21301, 430, 285, 0, RSinterface);

Method for setBounds();

public static void setBounds(int ID, int X, int Y, int frame, RSInterface RSinterface) { RSinterface.children[frame] = ID; RSinterface.childX[frame] = X; RSinterface.childY[frame] = Y; }

430 = X coordinate that is centered over the 123 - 128

Hope this helps anyone else who has the issue.
