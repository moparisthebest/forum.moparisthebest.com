A compile file is when you basically make your .class files for what the .java files are. But, the .class files aren‚Äôt supposed to be edited. The .java files are. You do not open a .dat file with any text editor. Open it up with a model viewer such as dat maker. The ONLY thing you really should edit, are the .java files.
