isnt it 995 for coins?

you use spaces after the pickup

so for example.

::pickup 0995 1000

bearing in mind if the item id is under 1000 you have to put a 0 infront

the 1000 is the ammount of that item you want, will only work if your spawning noted items otherwise it will just spawn one.

so if i wanted one abby whip i would enter this command.

::pickup 4151 1

normally for that noted item it is 1 number after the unnoted item

un noted abby whip = item code 4151
noted abby whip = item code 4142

it isnt always like this but most times it is,

thats only for items non stackable
