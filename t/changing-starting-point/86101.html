Go into player.java and search for // the first call to updateThisPlayerMovement() will craft the proper initialization packet

Underneath that you should see something like this // the first call to updateThisPlayerMovement() will craft the proper initialization packet teleportToX = 3489;//3489; teleportToY = 3288;//3288;

Change the teleportToX & teleportToY to the coords you wan’t to spawn at. Remember to only change the 1st two.

teleportToX = 3489;//3489;
teleportToY = 3288;//3288;
