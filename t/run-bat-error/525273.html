Fire_capez August 4, 2016, 9:46am #2

looks like incompatible jdk and jre versions, the jre runtime runs the bytecode the jdk compiles which in this instance the code has been compiled with a version of jdk higher then jre

basically, make sure jdk and jre are at the same version and upgrade them to a higher version if they are (if you can) if all else fails recompile it.

as for eclipse it does have a feature called compiler compliance level (window - preferences - compiler: set compiler compliance level to the version e.g 1.6)
