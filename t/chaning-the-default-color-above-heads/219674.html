Step 1:

Open client.java and search for

if(s.startsWith("yellow:"))

You will see something like

if(s.startsWith("yellow:")) { j2 = 0; aString887 = aString887.substring(7); }

And that short code makes the function yellow: work. ( Talking in yellow )

Step 2:

Now add this under it:

if(s.startsWith("")) { j2 = 0; }

What that does is that it makes the Default talk color change, without using functions like cyan:.

You can change the

j2 = 0;

to what ever you want. For exampel ;

j2 = 4;

Would be purple.
