Absolutely not. A java project can achieve multiple purposes, only one of which is an executable program by the JVM (for only those require a main method as an entry point). A non-exhaustive list of other uses for Java projects would include: APIs, mobile developments (dalvik apps have a different entry point than sun jvm apps), applets, java agents, and much more. Don‚Äôt think of a Java project as a program. Programs are just a subset of what Java projects can be used for.
