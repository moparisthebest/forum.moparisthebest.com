Ahh i’ve solved the problem!

Aha! This is such a strange error we could have been here forever,

In player.java

[code]str.writeBits(14, id); // client doesn’t seem to like id=0

	int z = npc.absY - absY;
	if (z < 0)
		z += 32;
	str.writeBits(5, z); // y coordinate relative to thisPlayer
	z = npc.absX - absX;
	if (z < 0)
		z += 32;
	str.writeBits(5, z); // x coordinate relative to thisPlayer

	str.writeBits(1, 0); // something??
str.writeBits(12, npc.npcType);[/code]

I had the npc.npctype at 14 and the only one that the client don’t like at 12, I need to reverse them.
