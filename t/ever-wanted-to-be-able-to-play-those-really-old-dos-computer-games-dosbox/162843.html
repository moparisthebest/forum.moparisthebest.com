Part1

  1. Download DOSBox 0.72 from Their Homepage

  2. Make a folder on your C Drive, call it DOS

  3. Run the DOSBox 0.72 installer, change the installation directory to your newly created folder.

  4. After its completed, go to your DOS folder.

  5. Create a new folder inside the DOS folder, call it CDRIVE.

Part2

  1. Open up dosbox.conf in notepad.

  2. Scroll to the bottom of the document, find this.

[autoexec]
# Lines in this section will be run at startup.
  1. Open up My Computer, find your disk drive letter

Example: Mine would be E

  1. Go back to the dosbox.conf file and add these lines under

Lines in this section will be run at startup.

mount C C:\DOS\CDRIVE
mount * *:\ -t cdrom (REPLACE * WITH YOUR DRIVE LETTER!!!!!)
C:
CLS

For example mine would look like this:

mount C C:\DOS\CDRIVE
mount E E:\ -t cdrom
C:
CLS

After putting that in, save the dosbox.conf file, close it.
