Hello everyone,

I am releasing another Open_RSCD Webclient! This one includes regular, large, and huge size clients, and an updated README guide.

It has been designed to support several of the Noobscape3 functions such as right click add player, right click kick, teleport, summon, along with the ::staff command to telewalk by clicking on the map. (Admins only)

The quests tab is compatible with Open_RSCD_v4/Legendscape (soon to be released) but can be made compatible with Noobscape3 by changing the packet IDs associated with each quest inside the mudclient.java file.
