Hi guys. Those of you that use the IRC channels may have met TimGroe, a mate of mine. Anyway, he has been working on a website, called ShareSource, designed for developers like yourselves to organise and host open-source software projects.

ShareSource can be found at http://sharesource.org/

ShareSource has several features:

[ul][li]Free subversion hosting for your projects[/li]
[li]A bug tracker, which allows the submission of bugs found by users of your software[/li]
[li]A version release system, which shows a snapshot of every version made to date[/li][/ul]

Signing up is very easy. Setting up a project is even easier. Project setup is achieved in just 4 simple steps. You can enable/disable features like the bug tracker or SVN snapshot page if you choose to. The only information required from you when creating a new project is the name of the project, a description of the project, and the type of project (basically what it’s related to. eg: databases, networking).

There is only one requirement on your part - your project must be open-source, and it must use one of the open-source licenses listed below.
