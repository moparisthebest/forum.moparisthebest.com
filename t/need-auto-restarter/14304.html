lets say this is my compile.bat (it is):

@echo off
COLOR 08
title Martinscape Client
"C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java
"C:\Program Files\Java\jre1.5.0_07\bin\java.exe" server
pause

ok in your compile.bat put in behind server this:

 -Xmx500m

so the whole compile should look lsomething like this:

@echo off
COLOR 08
title Martinscape Client
"C:\Program Files\Java\jdk1.5.0_07\bin\javac.exe" -cp . *.java
"C:\Program Files\Java\jre1.5.0_07\bin\java.exe" -Xmx500m server
pause

This prevents your server from crashing so you dont need to worry about a auto-restarter!

Please dont copy the whole damn thing into your compile.bat as everyone has different path, if so than copy it but change it to your old path.
