Weibs July 30, 2016, 2:37am #15

Yup, look at what cruel posted.

I’ll break it down, too:

NewItemString := copy(ItemString, p+Length(t[i])+1, (Length(ItemString)-Length(t[i])+1));

Now, first, I call copy(), and the first parameter

copy(ItemString,

Is the String I’m copying from. Next, the parameter calls for the starting point within the String to begin copying, so I tell it this:

p+Length(t[i])+1

P holds the position of the first letter of one of the words (Wear, Wield, or Use). Then, it adds the length of the word it found

+Length(t[i])

So that it doesn’t copy the word which you are trying to trim out, and then it adds 1 to skip the space after the word.

The next parameter calls for a count of how many characters to copy starting from the position we just gave it.

This is what we have:

(Length(ItemString)-Length(t[i])+1)

First we get the length of the original item string (ItemString):

(Length(ItemString)

and subtract from it the length of found word(Wear, Wield, or Use). It does this because we are starting the copying after that word, and then we add one to that count for the space.

I hope that cleared it up,

-Weibs.
