* Ability to see how much exp you need to get to your next lvl.

* Compiles in about 150 msc.

* Add or delete as many player names as you want easily. Load
users from a text file or save usernames to a text file.
* Calculates combat and your combat class. 99% accurate. Any info?
* To Do list for stuff you need to do.
* Load stats from an INI file and if you load stats on both sides
