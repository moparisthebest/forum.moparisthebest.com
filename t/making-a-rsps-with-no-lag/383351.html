ry60003333 August 1, 2016, 7:43am #17

First off, if you don’t want any lag, you’ll need a server that is actually written correctly, such as Hyperion. Most of the servers you find here, regardless of the client version they use, will lag horribly when more players start to join. Also, their CPU and RAM usage will probably skyrocket, due to poor programming and memory leaks.

However, since I am assuming that you are using one of these servers, you’ll want a computer, or a VPS, with enough RAM and a powerful enough CPU to handle however many players who get. If you are running it from home, you’ll also want a good internet connection.
