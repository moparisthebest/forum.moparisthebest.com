Step 1:
First create a new stranded exe document.

Now make 4 command buttons.

cmdBack = Back.
cmdForward = Forward.
cmdHome = Home.
cmdStop = Stop.

and if you want you can make one more.

cmdSearch = Search.

Step 2:
Open Visual Basic, and Press CTRL+T or go to the Project Menu -> Components. When the Components dialog pops-up scroll down to the Microsoft Internet Controls in the Controls section, select it and press OK.

You should now see a globe on your toolbox now. if not repeat step 2 and read it more carefully.

Step 3:
Create one label and name it: lbCaption
Create one combo box control and name it: cboURL
and with the web browser control you‚Äôve just added, and name it: wWeb

mess around with it and now it should turn out like this:

Step 4:
Now that you‚Äôve finished the UI [User Interface] for your web browser, here comes the easy/hard yet fun part of making your browser! The coding , you have to write codes for your browser to function properly.

Double Click cmdBack then type this code in:

wweb.GoBack

the GoBack method makes the web browser control to go back to the previous page.

Double Click cmdForward then type this code in:

wWeb.GoForward

The GoForward method makes the web browser control to go forward to another page.

Double Click cmdStop then type this code in:

wWeb.Stop

The Stop method makes the web browser control stop whatever it‚Äôs doing.

Double Click cmdHome then type this code in:

 wWeb.GoHome

The Home method makes your web browser control navigate back to the webpage that you set as home in the Internet Explorer Option.

Then Right click anywhere on your form then click view code and put this in at the bottom:

cboURL_KeyDown() If KeyCode = vbKeyReturn Then wWeb.Navigate cboURL.Text End If
That code will make the web browser control navigate to the URL you that you specified in the cboURL whenever you press the return key while typing in the combo box.
