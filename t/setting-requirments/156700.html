Armour and weapons requirements are defined in an int.

The ints are:

Code:

public int GetCLAttack(int ItemID) {
public int GetCLDefence(int ItemID) {
public int GetCLStrength(int ItemID) {

and so on…

Adding an item to one of these ints can be done in 2 ways. You can use item id’s as i will show you in the next stage or you can use itemNames as i will show you in this stage.

To make a requirement for an item name you need to add this under one of the ints of your choice.

Code:

if (ItemName.startsWith("ITEMNAME")) { return ##;

The ITEMNAME is the item name of the item. So im going to use dragon as my choice.

The return ## is the level required to wear the item.

So for my dragon im going to do 60 defence to wear, so under

public int GetCLDefence(int ItemID) {

im going to add

} else if (ItemName.startsWith("Dragon")) { return 60;

That means any item starting with the name dragon will require 60 defence. But dont worry, you can still make weapons 1 defence and i will show how in the next stage.

You can code requirements by using item id’s aswell as names. In this stage you will learn how to make items like dragonhide only need one defence.

Search for the int you want to use. So im going to show how to make a dragon 2h which is normally 60 defence require only 1 defence.

So i search for

if (ItemID == 7158) { return 1; }

Now that makes it so you only need 1 defence to use a dragon 2h.

You can use this method to make it so that you only need 1 defence to wield dragonhide too.

You can use almost anything to make a new requirement. From total level to quest points!

You will need to set up a new int

Making a new int isnt hard at all.

Just search for

int CLAttack = GetCLAttack(wearID);

Im going to show how to use total level as a requirement.

And add this under it
