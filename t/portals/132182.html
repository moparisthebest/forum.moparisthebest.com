Sick_of_Air July 30, 2016, 1:30pm #2

case OJBECTID: if(GoodDistance2(absX, absY, objectX, objectY, 3)) { if(actionTimer = 0) { teleportToX = XCOORDS; teleportToY = YCOORDS; actionTimer = 10; updateRequired = true; appearanceUpdateRequired = true; } } break;
Replace OBJECTID with the portal’s ID; XCOORDS with the X Coordinates that you want; and YCOORDS with the Y Coordinates that you want.
