Well the domain of a function is the set of all real numbers that can be applied for x that ensure that f(x) results in a real value; the key points too look out for in basic problems like this are possible negative square roots and divisions by zero. The range of a function is the set of all real numbers that f(x) can result with. As your function is a linear equation, the range is also the set of all real numbers. The main thing you need to look out for here is powers of multiples of 2 (^2, ^4, etc) since they result in only positive values.
