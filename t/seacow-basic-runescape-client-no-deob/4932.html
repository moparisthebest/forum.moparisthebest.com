Lol, wget IS a program, i putted it into the seacow_files folder :P.
And the account creator is a java file made by Iamadam.
Well, it is somehow ripped by Aryan, but edited by iamadam for ARGA and i could use it.

I dont want to gice you the source, but you can get it by using DJ Java Decompiler with NewChar.

And if you want to use the character creator alone, make a batch file like this:

@ECHO OFF
TITLE Account Creator
ECHO Welcome to the Account Creator by Iamadam.
JAVA NewChar
ECHO Thanks for using the Account Creator.
PAUSE

and save it as Acc.Creator.bat or whatever (it has to end with .bat) and put the NewChar.class (find in the Seacow_files folder) in the same folder as the batch file.
