sudo ./jdk-6-linux-i586.bin

Press space a few times then type yes (it may just be y).
Now continue by typing this:

cd <the jdk folder name>

sudo tar -xvf jdk.tar
sudo rm jdk.tar

Congrads you have installed JDK 1.6, you may delete the original jdk folder and .bin if you like.
