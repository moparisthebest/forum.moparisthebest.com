inferno July 29, 2016, 8:12pm #4

Well if you did use the movemouse and clickmouse you would need to put

if(FindBitmap(Bitmap, X, X))Then begin MoveMouse(X, Y); Wait(200+ Random(100)); ClickMouse(X, Y, True); end;

Because if you didn’t have the begin and end then it would perform the clickmouse command even if it didn’t find the bitmap and click at the previous values of the x and y variables, which you don’t want to do.
