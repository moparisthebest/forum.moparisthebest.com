Fridder August 4, 2016, 1:40am #5

To get rid of the ‘bits of grass’, in your client class, search for ‘setHighMem()’ and you should see:

	private static void setHighMem() {
		WorldController.lowMem = false;
		Texture.lowMem = false;
		lowMem = false;
		ObjectManager.lowMem = false;
		ObjectDef.lowMem = false;
	}

Now you should see:

ObjectManager.lowMem = false;

You want to change that to ‘true’, so it would look like this:

	private static void setHighMem() {
		WorldController.lowMem = false;
		Texture.lowMem = false;
		lowMem = false;
		ObjectManager.lowMem = true;
		ObjectDef.lowMem = false;
	}
