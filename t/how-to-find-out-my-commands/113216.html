Well, if you mean inside your own server source, you open up the folder containing client.java then you open client.java and search(ctrl+f) for command until it says something similar to this:

if(command.equalsIgnoreCase("combatz"))

The command is the the word in between the quotes(i.e. combatz), you then start the server and input two colons (: then the command so it looks like this:

::combatz
